                        Codenvy Platform SDK
                       ------------------------
                       Build Your Own Cloud IDE


=============
Requirements
=============

 *  Java Platform, Standard Edition 7 Development Kit (JDK 7) or later;
 *  Apache Maven version 3 or later.


===========
Installing
===========

1. Download and install JDK 7.

1.1. Download JDK, release version 7 or later, from
     http://www.oracle.com/technetwork/java/javase/downloads/index.html.

1.2. Install JDK according to the instructions included with the release.

2. Download and install Apache Maven.

2.1. Download Apache Maven, release version 3 or later, from
     http://maven.apache.org/download.cgi.

2.2. Install Apache Maven according to the instructions included with the release.

3. Make sure JAVA_HOME is set to the location of your JRE.
   Make sure M2_HOME is set to the location of your Maven.

4. Download and unpack Codenvy Platform SDK.

4.1. Download a binary distribution of Codenvy Platform SDK.

4.2. Unpack the archive where you would like to store the binaries.
     A directory called "codenvy-platform-sdk-[version]" will be created.

5. Codenvy Platform SDK requires some libraries in order to launch Codenvy-extensions that you develop.
   You should to copy these libraries to your local Maven repository.

5.1. Execute the following command:

On *nix:
      install.sh path
      , where path is a path to your local Maven repository.


===================
Running & Stopping
===================

1. In order to launch a Codenvy Platform SDK, go to bin directory and start Tomcat:

On *nix:
      ./catalina.sh start

After launching, Codenvy Platform SDK application will be available on http://localhost:8080/

2. To stop a Codenvy Platform SDK application, execute the following command:

On *nix:
      ./catalina.sh stop


====================================================
Building Tomcat bundle with your own Codenvy plugin
====================================================

1. In order to build Tomcat bundle that contains Codenvy with your extension, go to 'Project' -> 'Build Tomcat bundle'.

2. Once build is complete the following message will be displayed in 'Console' view:
      You can get your Tomcat bundle here: /<path_to_your_codenvy_sdk>/codenvy-platform-sdk-<version>/temp/codenvy-sdk-tomcat-bundle.zip

3. Take this zip and unpack it to the desired location for a Tomcat bundle:
      unzip codenvy-sdk-tomcat-bundle.zip

4. You need to change permissions for all sh-scripts in the 'bin' directory to executable:
      chmod +x bin/*.sh

5. You can start/stop Tomcat bundle by executing an appropriate command:
      ./catalina.sh start
      ./catalina.sh stop
